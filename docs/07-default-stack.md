# Default Stack (works for most apps)

## Frontend
- React/Next.js + TypeScript
- React Query / SWR
- Zod for schema validation

## Backend
- Node (Nest/Express/Fastify) or Python (FastAPI)
- OpenAPI / typed contracts

## Database
- Postgres + migrations (Prisma/Drizzle/Flyway/Liquibase)

## Auth
- Cookie sessions OR short-lived JWT access + refresh rotation

## Storage
- S3-compatible buckets + CDN

## Cache/Queue
- Redis + job worker

## Observability
- Structured logs + metrics + tracing

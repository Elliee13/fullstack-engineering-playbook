# Architecture Decision Records (ADRs)

ADRs capture important architectural decisions in a lightweight, reviewable format.

## How to use
1) Copy `0000-template.md` to `000X-<short-title>.md`
2) Fill it out and open a PR
3) Link relevant issues/specs
4) Keep decisions short and factual
